## Vercel 배포

압축을 푼 뒤 파일 전체를 GitHub 저장소 최상위에 올리고 Vercel에서 Import합니다.

- Framework Preset: Other
- Root Directory: 비워두기
- Build Command: 비워두기
- Output Directory: 비워두기
- Install Command: 비워두기

점수는 브라우저 localStorage에 저장되며, 1탄 기록과 별도로 관리됩니다.
