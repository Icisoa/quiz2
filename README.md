# 똘계 나트랑 일정 퀴즈 2탄 — 공용 순위표 버전

Supabase 프로젝트 URL과 Publishable key가 이미 연결되어 있습니다.

## 최초 1회 Supabase 설정

Supabase → SQL Editor → New query에서 `SETUP_SUPABASE.sql` 내용을 전부 붙여넣고 Run을 누릅니다.

## Vercel 배포

파일 전체를 GitHub 저장소 최상위에 올리고 Vercel에서 Import합니다.

- Framework Preset: Other
- Root Directory: 비워두기
- Build Command: 비워두기
- Output Directory: 비워두기
- Install Command: 비워두기

모든 참가자의 이름과 점수가 동일한 Supabase 공용 순위표에 저장됩니다.
